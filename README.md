# 🏀 NBA2KOL2 智多星活动全自动AI答题助手

[![Version](https://img.shields.io/badge/version-1.0-blue.svg)](2k.user.js)
[![Tampermonkey](https://img.shields.io/badge/Tampermonkey-Supported-green.svg)](https://www.tampermonkey.net/)

这是一个为 NBA2K2 OL 设计的油猴脚本（UserScript），能够全自动识别网页上的答题内容，并调用 AI 模型（如 DeepSeek、OpenAI）进行自动答题。

## ✨ 主要功能

- **自动化流程**：自动点击“开始答题”按钮，无需手动干预。
- **智能识别**：自动抓取题目和选项内容。
- **AI 加持**：支持配置 OpenAI 兼容接口（推荐使用 DeepSeek），利用大模型高准确率回答问题。
- **伪装环境**：内置 User-Agent 伪装，模拟游戏客户端环境，防止因环境检测导致的无法答题。
- **可视化面板**：提供直观的配置界面，可实时修改 API 地址、API Key 和模型名称。
- **答题记录**：(如果脚本有记录功能) 记录答题历史，方便回顾。

## 🛠️ 安装说明

1. **安装浏览器扩展**：
   确保你的浏览器（Chrome, Edge, Firefox 等）已安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/) 插件。

2. **安装脚本**：
   - 如果你有脚本文件，直接拖入浏览器扩展管理界面安装。
   - 或者点击 [这里](2k.user.js) (如果托管在 GitHub/GreasyFork) 安装。

## ⚙️ 配置方法

脚本初次运行或点击界面上的设置按钮时，你可以配置以下内容：

- **API Host**: AI 服务的接口地址 (例如: `https://api.deepseek.com/v1`)。
- **API Key**: 你的 API 密钥 (例如: `sk-xxxxxxxx`).
- **Model**: 使用的模型名称 (例如: `deepseek-chat`).

> **注意**: 默认配置推荐使用 DeepSeek API，性价比高且中文理解能力强。

## 🚀 使用方法

1. 打开 NBA2K2 OL 活动页面 (匹配 `https://nba2k2.qq.com/*`)。
2. 脚本会自动加载，并在页面显示操作面板（通常在右侧或左侧）。
3. 确保 API 配置正确。
4. 当活动开始时，脚本会自动检测并开始答题流程。

## 📚 题库管理功能

脚本内置了强大的本地题库管理系统，支持自动记录和手动管理。


### 1. 自动记录
在答题过程中，脚本会自动将**题目**、**选项**和**AI 回答的正确答案**保存到本地存储中。下次遇到相同题目时，脚本会优先检索本地题库，提高答题速度和准确率。

### 2. 题库编辑器
点击面板上的 **“📝 编辑题库”** 按钮即可打开管理器：
- **搜索题目**：输入关键词快速查找历史题目。
- **编辑/修正**：点击题目展开详情，可手动修改题目文本或修正错误的答案。
- **删除记录**：对于不再需要的题目，可以单条删除。

### 3. 导入与导出
方便你在不同设备间同步题库，或分享给通过好友：
- **导出**：将当前积累的题库保存为 `.json` 文件。
- **导入**：选择 `.json` 题库文件进行导入（注意：导入会覆盖当前题库）。
